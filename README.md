## Compatible Components

These cases are designed for **Smol Stacked SlimeVR** builds using:

- **MCU:** SuperMini NRF52840
- **IMU:** Various supported sensors (primarily ICM-45686)
- **Battery:** Small LiPo cells appropriate for case dimensions. **401230** Refer to last IRL image to see the space.

## Print Settings

| Setting | Value |
|---------|-------|
| Layer Height | 0.2mm |
| Infill | 15–20% |
| Supports | Automatic is OK |
| Material | PLA or PETG |

### Tips

- Print a test fit before mass printing
